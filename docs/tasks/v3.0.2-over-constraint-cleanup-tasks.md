# v3.0.2 Over-Constraint Cleanup — Task Document

## Context

Implements [docs/plans/v3.0.2-over-constraint-cleanup.md](../plans/v3.0.2-over-constraint-cleanup.md).

Removes two v3.0.0-introduced over-constraints that violate v3's own
bitter-lesson philosophy:

1. Forced English output across 4 agent files + project `CLAUDE.md` +
   plugin `README.md` + `plugin.json` description + retire v3 plan AC6.
2. `Status` column in Planned Dispatch tables across 5 dispatching
   `SKILL.md` files + reverse v3 plan AC8.

`plugin.json` version bumps `3.0.1 → 3.0.2`.

Single-phase plan (no Phase 1/2 split). Tasks 1–5 are independent;
Task 6 depends on Tasks 1–5 (AC checks in Task 6 require all prior
edits to be in place).

Related plan: [docs/plans/v3.0.2-over-constraint-cleanup.md](../plans/v3.0.2-over-constraint-cleanup.md)
Upstream spec: [docs/plans/v3-bitter-lesson-refactor.md](../plans/v3-bitter-lesson-refactor.md)
